### Getting Started

- Step 1: Clone repository.
  - SSH: `git@gitlab.com:seven-tech-01/hr-management.git`.
  - HTTPS: `https://gitlab.com/seven-tech-01/hr-management.git`.

- Step 2:
  - Open terminal: `cd .\hr-management\`
  - Install the packages `pnpm install`.

- Step 3:
  - Add file .env and add elements like .env.example
  - Run app:
    - Run web local: `pnpm dev`
    - Run json server: `pnpm start`
